# ChatRoomRecorderReborn

Multiple websites - Supports the most popular Chaturbate, BongaCams, Stripchat, Flirt4Free, CamSoda and Cam4 websites
No limitations - Records as many streams as you want
Automatic recording - Automatically starts recording when streams are online
Open source - Released under GNU GPLv3
Installation instructions
Download and install .NET Desktop Runtime (x64 or x86) and WebView2 Runtime (x64 or x86)
Download and install Streamlink (x64 or x86)
Download and unpack ChatRoomRecorder (x64 or x86)
Start ChatRoomRecorder
Open "Settings" tab and specify Streamlink and ffmpeg binaries locations and an output directory for downloads
Open "Web browser" tab, visit supported web sites and pass bot protection
Open "Chat rooms" tab, add desired links and start recording
